##### 1、复制除eval外其他所有JS代码
##### 2、打开chrome开发者模式，并切换到console标签
##### 3、粘贴js代码，敲回车，即可得到明文JS
##### 4、使用在线代码格式化，格式化JS代码:[在线格式化代码](http://tool.oschina.net/codeformat/js/)
